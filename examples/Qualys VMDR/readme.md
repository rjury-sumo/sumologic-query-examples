# Qualys VMDR
## Sumo Logic App For: Qualys VMDR
Qualys provides cloud security, compliance, and vulnerability management solutions. The Sumo App for Qualys VMDR provides vulnerability summary across your IT infrastructure. The App dashboard gives you visibility into low, medium, high, and critical vulnerabilities by hosts in your network. The App filters allow you to slice and dice data by vulnerability severities, IPs, and hosts.
Docs Link: [Qualys VMDR](https://help.sumologic.com/?cid=97652)
